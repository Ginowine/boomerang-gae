GDG Space Coast - Intro to Dart and How to fix Git

TrepHub, 907 Strawbridge Ave, Melbourne, FL
Thursday, December 5, 2013 - 7:00 PM

Primary Topics: Introduction to Dart. How to fix Git.

Dart is a web programming language by Google. It compiles to JavaScript, but has a lofty goal of replacing JavaScript. It aims to provide much more powerful tooling and performance compared to JavaScript. It can run native inside the custom Dartium browser. The Dart team just announced their 1.0 release this month!

Git is a free version control system that runs on most all platforms and is becoming incredibly popular. Both GitHub and Bitbucket support collaboration in Git. Most of the time it is fairly straight forward. But when problems occur, it is sometimes infuriating to try and solve them. In this presentation, you will be given some tools for dealing with those kinds of issues!

GCDC: GDG Space Coast: App Engine Code Camp #3

TrepHub, 907 Strawbridge Ave, Melbourne, FL
Saturday, October 12, 2013 - 1:00 PM

Bring your laptops! This is the third in a series of events focused around the #GCDC 2013! Find out more about Google Cloud Developer Challenge 2013 and the $20,000 grand prize here: http://www.google.com/events/gcdc2013/

The focus of the code camp is learning and not competition. There will be people there to help and answer questions as you walk through the steps to build a web application including things like Queues, Backends, Warmup Requests, and Pagination.

Each of the 5 tutorials will build upon the previous tutorials.
